Editor [General Formulas]

In the "General Formulas" tab, you can configure a list of general formulas that can be substituted and used within various formulas and conditions. Up to 999 can be configured.
Each general formula can be written in the form of formula[ ] (formula[0] to formula[999]) within various formulas and conditions, and will be replaced when the game data is loaded in the main program. It can be used in the same way as reference values.

  • search
    This is a search button used when there are many items in the list. It searches for general formulas containing the string written in the text field.
  • Name
    This is set by the scenario creator for identifying each general formula and is not used during the game. Leaving it blank is acceptable.
  • Formula
    Write the formula here. Line breaks are ignored during substitution.

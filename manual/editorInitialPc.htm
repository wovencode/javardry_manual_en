Editor [Initial PC]

In the "Traps and Initial PC" tab, you can configure the list of initial PCs. Initial PCs are adventurers registered in the training grounds upon the first startup or initialization of Javardry, and up to 120 can be set (the maximum number of registrations in the training grounds can be set in the "PC Creation and Job Change" tab). Initial PC configurations might not be used frequently, so it is fine to leave them unset if it is too much trouble.
When adding PCs through events, you will specify the PCs configured here.

  • Do not place initially
    Checking this will prevent the PC from being placed during the first startup (initialization).
  • PC Name
    The name of the adventurer. Up to 20 characters in half-width.
  • Gender, Alignment, Race
    Select gender, alignment, and race. If the selected gender conflicts with the gender settings of the race, the race settings will take precedence, and the gender will be adjusted accordingly.
  • Job
    Choose the adventurer's job from the job list. Please note that jobs can be set here regardless of gender or alignment requirements.
  • Attributes
    Specify the adventurer's attributes with values between 0 and 999. Clicking the "set" button will apply the value of the first attribute to all attributes.
  • Level
    Specify the adventurer's level using a formula.
  • Experience Points
    Specify the adventurer's experience points between 0 and 999999999999. If the specified value is insufficient for the required experience points of the given level, it will be corrected. For simplicity, you can set it to 0.
  • Gold
    Specify the amount of gold using a formula.
  • Age
    Specify the adventurer's age between 1 and 9999999. If you want to determine the age using the same formula as PC creation, set it to 0.
  • Title
    Specify the adventurer's title as a string of up to 6 half-width characters.
  • Items
    Specify the items the adventurer possesses by their item numbers. Multiple items can be specified separated by commas, but any exceeding the maximum carry capacity determined by the race and job will be ignored.
  • Equip
    Checking this will allow the adventurer to start with equippable items already equipped.
